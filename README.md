# Pixel Invaders – Suporte (site estático)

Repositório ao lado do projeto do jogo (`../pixel-invaders`) para hospedar a página de suporte em HTML.

- **Conteúdo:** `index.html` (página de suporte; pode ser publicada em GitHub Pages ou outro host estático).
- **Jogo (Godot):** pasta irmã `pixel-invaders`.

## Publicar no GitHub

1. Crie um repositório vazio no GitHub (por exemplo `pixel-invaders-support`).
2. Adicione o remoto e envie os commits:

```bash
cd /Users/rafaelrosa/games/pixel-invaders-support
git remote add origin git@github.com:SEU_USUARIO/pixel-invaders-support.git
git push -u origin main
```

3. Em **Settings → Pages**, escolha a branch `main` e a pasta `/ (root)`.
